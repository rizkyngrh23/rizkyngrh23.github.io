Working with Databases

Databases are essential for storing and managing data in modern applications. In this article, we'll explore how to work with databases using Node.js. We'll cover setting up a database, connecting to it, and performing basic CRUD (Create, Read, Update, Delete) operations.

Step 1: Choose a Database

There are many databases to choose from, including relational databases like MySQL and PostgreSQL, and NoSQL databases like MongoDB. For this example, we'll use MongoDB, a popular NoSQL database.

Step 2: Set Up MongoDB

First, you need to install MongoDB on your computer. Follow the installation instructions on the official MongoDB website. Once installed, start the MongoDB server by running:

mongod

Step 3: Connect to MongoDB

Next, we'll connect to MongoDB from our Node.js application. Install the MongoDB Node.js driver by running:

npm install mongodb

Create a new file called database.js and add the following code to connect to MongoDB:

const { MongoClient } = require('mongodb'); const uri = 'mongodb://localhost:27017'; const client = new MongoClient(uri, { useNewUrlParser: true, useUnifiedTopology: true }); async function connect() { try { await client.connect(); console.log('Connected to MongoDB'); } catch (err) { console.error(err); } } connect();

Step 4: Perform CRUD Operations

Now that we're connected to MongoDB, let's perform some basic CRUD operations. Add the following code to database.js:

async function performCRUD() { const db = client.db('mydatabase'); const collection = db.collection('mycollection'); // Create await collection.insertOne({ name: 'John Doe', age: 30 }); console.log('Document inserted'); // Read const document = await collection.findOne({ name: 'John Doe' }); console.log('Document found:', document); // Update await collection.updateOne({ name: 'John Doe' }, { $set: { age: 31 } }); console.log('Document updated'); // Delete await collection.deleteOne({ name: 'John Doe' }); console.log('Document deleted'); } performCRUD();

Conclusion

In this article, we've covered the basics of working with databases in Node.js using MongoDB. We've set up a MongoDB server, connected to it from our Node.js application, and performed basic CRUD operations. From here, you can explore more advanced topics like indexing, aggregation, and transactions.

Happy coding!
